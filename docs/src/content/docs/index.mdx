# @f0rbit/runbook

Typed workflow engine for orchestrating AI agents, shell commands, and human checkpoints.

## Features

- **Type-safe pipelines** — Zod schemas at every boundary
- **4 step types** — fn(), shell(), agent(), checkpoint()
- **Agent orchestration** — dispatch to AI coding agents with typed I/O
- **Composable workflows** — pipe(), parallel(), asStep()
- **In-memory testing** — Provider pattern replaces all external I/O
- **Git artifact store** — trace storage in git refs
